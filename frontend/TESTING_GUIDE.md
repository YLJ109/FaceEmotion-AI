## 🐛 常见问题排查

### 问题 1：滚动条未显示
**可能原因**：
- 浏览器不支持 `::-webkit-scrollbar`（如 Firefox）
- CSS 未正确加载

**解决方法**：
```bash
# 清除浏览器缓存并刷新
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)

# 检查控制台是否有 CSS 加载错误
F12 → Console 标签
```

### 问题 2：容器高度仍然过大
**可能原因**：
- 父容器设置了固定高度
- Flex 布局未正确配置

**解决方法**：
```css
/* 确保父容器使用 flex 布局 */
.parent {
  display: flex;
  flex-direction: column;
  height: 100%;
}

/* 子元素使用 flex: 1 自动填充 */
.child {
  flex: 1;
  overflow-y: auto;
}
```

### 问题 3：滚动条颜色不统一
**可能原因**：
- 某些组件使用了内联样式
- CSS 优先级问题

**解决方法**：
```bash
# 检查元素样式
F12 → Elements 标签 → 选中滚动条元素
# 查看 Computed 样式，确认是否被覆盖
```
